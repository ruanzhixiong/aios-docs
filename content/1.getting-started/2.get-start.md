### 1. 创建并启动插件项目

你需要先在本地创建一个插件项目，可以使用我们准备好的 [chat-plugin-template](https://github.com/rzx007/chat-plugin-template) 模板；

```bash
git clone https://github.com/rzx007/chat-plugin-template.git
cd chat-plugin-template
pnpm install
pnpm dev
```
